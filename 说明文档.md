该文档主要说明这个项目的内容：
项目主要是用docker对claude code 进行运行管理，相当于claude code的开发和生成在docker 中进行，但是将项目文件通过数据卷映射出来。

1.值得注意的是 docker中文件储存主要是在lunix 所指的地址，通常不是我们宿主机D/C盘这类，因此在处理时要特别注意生成项目存储的地址。
2.claude code 和项目开发的环境诸如python环境或者其他项目环境需要确定一下，因为claude code 可以对项目进行调试，但是docker 中的环境有限，如何进行互相通信修改非常重要，诸如claude code 在一个容器中，而项目的开发环境在另外一个新建的容器中。
